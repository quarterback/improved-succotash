The Occupant Index synthesizes pricing from multiple public sources into a single reference point, expressed as an index where February 2025 = 100. We publish daily using a standardized methodology designed for consistency.

Methodology

The Compute Price Index synthesizes publicly available pricing data into a volume-weighted basket across four workload tiers. We cross-validate across sources and flag discrepancies before publication.

Index Composition Weighting
$CPI Composite of all tiers Volume-weighted
$BULK Budget-tier models (<$1/MTok) Equal-weighted basket
$FRONT Frontier capability models Top-5 by Arena ELO
$JUDGE Reasoning-intensive workloads o1/o3/R1 class models
$LCTX Long-context workloads 128K+ context models

Build Cost Indices ($START, $AGENT, $THRU) apply fixed workload mixes to current tier prices, enabling organizations to track costs relevant to their specific use patterns.

Governance

The Occupant Index operates independently of model providers, cloud platforms, and API resellers. Index methodology changes are documented and disclosed prior to implementation.

The index does not accept sponsored placements, referral arrangements, or payments that could influence rankings or calculations.

Publications

Daily index values are published by 09:00 UTC. Periodic reports covering market movements, tier analysis, and cost forecasting are available to institutional subscribers.
